# ooad

# Command List
1. !keyword (word):
- 제목에 'word'를 포함한 공지를 올려드립니다. 
2. !trans (uri):
- 'uri'에 있는 공지사항 내용을 번역해서 유저에게 보냅니다.
3. !ques:
- admin의 email을 돌려줍니다.
4. !help:
- 가능한 명령어를 출력합니다. 
5. !sch (uri):
- 'uri'의 공지사항 일정을 디스코드 이벤트에 추가합니다.
